# Arithmetic
解析字符串形式的表达式，求解四则运算

要求： 
所有数字都是1-9的个位数 
运算符包括+、-、*、/、（、和） 
运算中间结果和最终结果为double类型

解题思路： 
1、常规解法（利用栈）：中缀转后缀，再求解； 
2、分治法（利用正则匹配）： 
1）只有加减运算时，从左往右算即可 
2）只有乘除运算时，从左往右算即可 
3）既有乘除也有加减时，按照“+”和“-”拆分，把乘除运算表达式看做一个整体（当做一个操作数，通过情形 2）算出） 
4）有括号时，先求解括号里面的表达式（按照情形 3）求解）； 
5）有括号嵌套时，从里向外循环求解（每一层都是情形 4））；
